---
title: usePageContent
description: Fetch one content document from a Nuxt Content page collection.
---

`usePageContent` fetches a single content document by path and collection. It defaults to the current route path and the `page` collection.

## Use it when

Use it in page components that need the shared content fetch but custom rendering.

```ts
const { data: page } = await usePageContent({
  collection: () => 'article',
  path: () => route.path,
})
```

The composable returns Nuxt `AsyncData`, so page components that need the first render to include content should `await` it in `<script setup>`.

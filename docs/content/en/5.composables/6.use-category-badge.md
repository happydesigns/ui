---
title: "useCategoryBadge"
description: "Resolve category badge configuration for article-like content."
---
`useCategoryBadge` reads category badge configuration from the selected collection variant and returns a computed badge object.

## Use it when

Use it in custom cards or metadata blocks that need the same badge configuration as `HArticleGrid`.

```ts
const badge = useCategoryBadge(() => article.category, 'article')
```

If a category is not configured, the composable returns a primary badge with the category key as its label.

---
title: "usePageSeo"
description: "Apply SEO metadata from content data."
---
`usePageSeo` reads SEO fields from content data and forwards them to Nuxt SEO metadata. It falls back from `seo.title` and `seo.description` to the document `title` and `description`.

## Use it when

Use it when a custom content page fetches data manually and should still use the same metadata behavior as shared pages.

```ts
const { data: page } = await usePageContent()

usePageSeo(page)
```

The composable watches the page ref, so metadata updates when the loaded document changes.

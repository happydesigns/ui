---
title: useArticleListQuery
description: Sync article list pagination and category filters with the route query.
---

`useArticleListQuery` keeps article list state in the URL. It is used by `HArticleList` and `HArticleGrid` so category and pagination changes can be shared, refreshed, or navigated with browser controls.

## Use it when

Use it when a custom list page still needs the same URL behavior as the shared article list components.

```ts
const { page, selectedCategory, updateQuery } = useArticleListQuery({
  labelAll,
  fixedCategory,
})
```

Call `updateQuery({ resetPage: true })` after changing the category. Call `updateQuery()` after changing the page.

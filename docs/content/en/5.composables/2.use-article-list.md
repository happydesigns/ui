title: useArticleList
description: Fetch paginated and filtered article-like content from Nuxt Content.
---

`useArticleList` is the data source behind `HArticleGrid`. It fetches one page of entries from an article-like collection, applies status/category/custom filters, sorts results, resolves category badges and authors, and returns a total count for pagination.

## Use it when

Use it when you need custom list rendering but still want the same query behavior as the shared article components.

```ts
const { data, status } = await useArticleList({
  collection: () => 'article',
  page,
  itemsPerPage: 12,
})
```

## Notes

- `collection` defaults to `article`.
- `lazy` defaults to `false` so list content resolves before rendering during normal navigation.
- `status` defaults to the configured status filter or `published`.
- `sort` defaults to the configured collection order.

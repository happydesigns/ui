---
title: "useClipboard"
description: "Copy text and optionally show a Nuxt UI toast."
---
`useClipboard` wraps VueUse clipboard behavior and adds optional Nuxt UI toast feedback. It is used by copy-link components and can be reused by project actions that copy text.

## Use it when

Use it when a project action should copy text and optionally show a toast through the shared Nuxt UI toast system.

```ts
const { copy, copied } = useClipboard()

copy(url, {
  id: 'copy-url',
  title: 'Link copied',
})
```

The returned `copied` ref comes from VueUse.

---
title: LLMs
description: Use Docus LLM output for @happydesigns/ui.
---

Docus exposes LLM-friendly output for the product docs. Use it when an agent needs a compact view of the documentation before editing code.

Useful endpoints after deployment:

| Endpoint | Use |
| --- | --- |
| `/llms.txt` | Compact documentation map. |
| `/llms-full.txt` | Larger documentation export. |
| Page copy button | Copy one page into an agent prompt. |

LLM output should be treated as documentation context, not as a substitute for checking source code.

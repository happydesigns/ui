---
title: "Skills"
description: "Install and use the happydesigns/ui agent skill."
---
The product docs publish a lightweight agent skill for work involving `@happydesigns/ui`. The skill points agents back to these docs instead of copying the documentation into a prompt file.

Use the skill when an agent edits:

- The Nuxt layer.
- Public `H*` components.
- Content schemas.
- Variants and app config conventions.
- Product docs for `ui.happydesigns.de`.

The skill should stay short. Product knowledge belongs in the docs; operational checks belong in the skill.

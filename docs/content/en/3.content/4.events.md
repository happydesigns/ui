---
title: Events
description: Use the event collection for dated entries with location and link support.
---

Use `event` for content that is primarily organized around a date and can include location or external link information. It shares much of the article model because events also need list pages, detail pages, categories, status, and optional authors.

## Typical setup

```vue
<template>
  <HArticleList collection="event" />
</template>
```

The article list components are generic over page collections, so the same list components can render events when the collection and variant configuration are set correctly.

## Event-specific fields

Events can use fields such as `dateEnd`, `location`, and `links` in addition to the common article fields. Use those fields to describe the event, not to control unrelated app behavior.

---
title: Snippets
description: Use snippets for reusable Markdown fragments.
---

Use `snippet` when a piece of content should be authored once and rendered inside another page or component.

```vue
<template>
  <HSnippet path="/snippets/contact" />
</template>
```

Snippets are useful for repeated contact blocks, notices, legal fragments, or editorial sections that should remain content-owned instead of hard-coded in Vue.

Do not use snippets as a configuration store. If the value changes component behavior, it belongs in app config or code.

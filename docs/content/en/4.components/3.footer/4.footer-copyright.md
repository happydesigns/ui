---
title: "FooterCopyright"
description: "Render project copyright text from app config."
---
`HFooterCopyright` renders copyright metadata from app config. Use it when the footer should stay configured rather than hard-coded in a page component.

The holder is rendered as plain text when `copyrightHomepage` is omitted. When a URL is configured, it is rendered as a regular link without forcing a new tab. Use the `copyright` slot for custom wording or link behavior.

---
title: "FooterButtons"
description: "Render a row of Nuxt UI buttons in footer content."
---
`HFooterButtons` renders configured Nuxt UI buttons. Use it for footer actions and social links when the project already has button definitions from app config or local data.

Use it when the footer needs a compact row of calls to action, social links, repository links, or legal links that should follow the same button conventions across projects. Keep project-specific labels and URLs in the project config or local footer data.

::ui-example{:preview='false' name="footer-buttons-basic"}
::

Buttons can be passed directly, but most project footers should read button data from app config or local footer data. Keep URLs and labels in the project; keep shared rendering behavior in the layer.

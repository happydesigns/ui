---
title: "FooterColumns"
description: "Render responsive footer columns from props or app config."
---
`HFooterColumns` wraps Nuxt UI footer columns and adds happydesigns layout behavior. It can receive columns directly or read footer columns from `app.config.ts`.

## Usage

::ui-example{:preview='false' name="footer-columns-basic"}
::

When `columns` is not passed, the component reads `app.links.footer` from `app.config.ts`. Use props for local previews or custom footer sections; use app config for the project-wide footer.

```ts [app.config.ts]
export default defineAppConfig({
  app: {
    links: {
      footer: [
        {
          label: 'Docs',
          children: [
            { label: 'Getting started', to: '/getting-started' },
          ],
        },
      ],
    },
  },
})
```

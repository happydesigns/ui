title: "ArticleList"
description: "Render category navigation and an article grid for an article-like collection."
---
`HArticleList` is the default list component for article-like collections. It reads collection category config, renders category navigation, syncs category state with the URL, and delegates result rendering to `HArticleGrid`.

Use it for blog, news, update, guide, or event index pages where the category filter is part of the route. Use `HArticleGrid` directly when the page already has its own navigation or should show only a single unfiltered grid.

::

The component keeps category state in the URL so links can open the same filtered view after navigation or reload.

### Fixed category

Pass `category` when the route should always show one category and hide the category navigation.

```vue [pages/articles/index.vue]
<template>
  <HArticleList collection="article" category="Release" />
</template>
```

### Custom card metadata

Slots are forwarded to `HArticleGrid`, so custom card metadata stays close to the list route while fetching and pagination remain shared.

```vue [pages/events/index.vue]
<template>
  <HArticleList collection="event">
    <template #date="{ article }">
      <HDates :date="article.date" :date-end="article.dateEnd" />
    </template>
  </HArticleList>
</template>
```

The public components are prefixed with `H` to separate happydesigns layer components from upstream Nuxt UI components. Use them when the project needs shared happydesigns behavior, content conventions, or app config integration.

## Component groups

| Group | Components |
| --- | --- |
| Article components | `HArticleList`, `HArticleGrid`, `HArticlePage`, `HArticleGridDate`, `HArticleHeaderBody`, `HArticleHeadlineMeta`. |
| Content components | `HSnippet`, `HCarousel`. |
| Footer components | `HFooterColumns`, `HFooterButtons`, `HFooterColumnHeading`, `HFooterCopyright`. |
| Trait components | `HAuthors`, `HBackButton`, `HCategory`, `HCopyButton`, `HDates`, `HLinks`, `HLocation`, `HSeparator`, `HSeparatorButtons`, `HSurround`, `HToc`. |

Generated API tables come from the actual Vue source. The prose explains when to use the component; the table explains the exact interface.

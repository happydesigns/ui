`HCarousel` wraps Nuxt UI carousel behavior for simple image galleries. Use it when a project needs a lightweight carousel from a list of image URLs.

## Usage

::ui-example{:preview='false' name="carousel-basic"}
::

Use `HCarousel` for straightforward image sequences where the project already has image URLs. Build a project-specific gallery when the carousel needs captions, custom media queries, mixed media types, or a content collection of its own.

```vue [components/GallerySection.vue]
<script setup lang="ts">
const items = [
  '/images/gallery/opening.jpg',
  '/images/gallery/event.jpg',
  '/images/gallery/team.jpg',
]
</script>

<template>
  <HCarousel :items="items" />
</template>
```

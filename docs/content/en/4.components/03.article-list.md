---
title: ArticleList
description: Render category navigation and an article grid for an article-like collection.
---

`HArticleList` is the default list component for article-like collections. It reads collection category config, renders category navigation, syncs category state with the URL, and delegates result rendering to `HArticleGrid`.

## Usage

:ui-example{name="article-list-basic" :preview="false"}

Use `HArticleGrid` directly when the page should not show category navigation.

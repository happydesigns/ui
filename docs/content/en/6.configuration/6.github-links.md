---
title: GitHub links
description: Configure source links for content-driven projects.
---

GitHub metadata lets content-driven pages point to their source repository. Use it when pages should expose edit or source links.

```ts
export default defineAppConfig({
  app: {
    meta: {
      github: {
        repo: 'happydesigns/ui',
        branch: 'main',
        dir: 'content',
      },
    },
  },
})
```

The values are project-specific. The layer should provide the config convention and source-link components; the project should provide the repository, branch, and content directory.

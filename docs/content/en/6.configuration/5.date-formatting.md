---
title: "Date formatting"
description: "Configure locale-aware date and datetime output."
---
Date display is controlled through `app.date`. Components such as `HDates` and article helpers use this configuration so project output stays consistent.

```ts
export default defineAppConfig({
  app: {
    date: {
      locale: 'en',
      options: {
        year: 'numeric',
        month: 'short',
        day: 'numeric',
      },
      datetimeOptions: {
        year: 'numeric',
        month: 'short',
        day: 'numeric',
        hour: 'numeric',
        minute: 'numeric',
      },
    },
  },
})
```

Use project config for locale and formatting choices. Keep date parsing and rendering helpers in the layer when the same behavior is reused.
